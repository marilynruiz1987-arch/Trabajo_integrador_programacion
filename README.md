Descripción del Proyecto

Este proyecto consiste en una aplicación desarrollada en Python para la gestión de datos de países. El sistema permite leer información desde un archivo CSV y realizar distintas operaciones sobre los datos almacenados.

La aplicación fue desarrollada aplicando los conceptos estudiados en la materia Programación 1, incluyendo listas, diccionarios, funciones, estructuras condicionales, estructuras repetitivas, manejo de archivos CSV, filtros, ordenamientos y estadísticas básicas.

Funcionalidades

El sistema permite:

- Agregar un país.
- Actualizar población y superficie de un país existente.
- Buscar países por nombre (coincidencia exacta o parcial).
- Filtrar países por continente.
- Filtrar países por rango de población.
- Filtrar países por rango de superficie.
- Ordenar países por nombre.
- Ordenar países por población.
- Ordenar países por superficie (ascendente o descendente).
- Mostrar estadísticas generales.

Instrucciones de Uso

1. Descargar o clonar el repositorio.
2. Verificar que el archivo paises.csv se encuentre en la misma carpeta que el programa.
3. Ejecutar el archivo principal:

python main.py

4. Seleccionar una opción del menú para utilizar las distintas funcionalidades.

Ejemplos de Uso

Buscar País

Entrada:

Argentina

Salida:

{'nombre': 'Argentina', 'poblacion': 45376763, 'superficie': 2780400, 'continente': 'America'}

Filtrar por Continente

Entrada:

Salida:

Lista de países pertenecientes al continente seleccionado.

Estadísticas

Salida:

- País con mayor población.
- País con menor población.

- Promedio de superficie.
- Cantidad de países por continente.
